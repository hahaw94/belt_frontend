智能视频分析平台 需求分析
登录功能
支持通过浏览器登录，在指定 IP 登录页面输入用户名和密码可登录系统。
首页功能
数据看板需显示相机在线数据、一周告警数据、告警分类排名、最新告警截图、CAD 图层，展示皮带业务场景并叠加摄像头，点击摄像头可弹出实时画面。
系统配置功能
基础管理：提供设备参数统一配置管理界面，支持前端设备和平台服务器自动时间同步，支持一键修改平台 IP，修改后自动重启服务，通过新地址可访问平台。
版本管理：支持备份系统版本，可上传 .zip、.tar、.gz 等压缩包格式对平台进行一键升级。
地图管理：支持平面布置图功能，可本地导入图片文件作为地图背景，可自行移动调整摄像机点位，调整后的点位信息能导出保存和再次导入，平台布置图替代远程电子地图展示在地图界面。
算法管理功能
算法仓：支持存储上传的算法模型，每个算法可存 5 个模型，支持手动上传和训练平台自动下发，删除模型需确认。
算法下发：可批量下发算法模型至智能分析板卡节点。
算法配置：支持为智能分析板卡的每一路视频设置规则，并同步至智能分析板。
用户管理功能
用户管理：支持单一添加或批量添加用户。
角色管理：默认有管理员和操作员两种角色，操作员可自定义操作功能模块，系统可自定义多个操作员角色。
接入管理功能
设备管理：可在门户对摄像机、智能分析节点进行安装配置，设备以列表视图呈现，支持手动添加和平台设备同步、单一添加和批量添加，智能分析板卡节点需与摄像机绑定关系并支持算法配置，相机接入支持 1000 路，需记录相机和智能板卡的多种设备信息。
设备接入协议：支持符合 GBT28181 - 2016 版本规范的视频摄像机接入，具备实时视频预览、录像和云镜控制功能，支持 rtsp 及 onvif 协议，最高兼容 2688×1520@30fps ，支持 H.264 及 H.265。
实时检测功能
支持对相机监控场景进行实时展示，可 4 分屏、9 分屏、16 分屏展示，实时画面需显示算法跟踪框及预警信息。
录像管理功能
系统需存储触发预警前后 10 秒的录像，录像画面包含跟踪框信息。
事件中心功能
告警信息展示：支持缩略图及列表形式查看告警信息，展示告警类型、时间、点位等信息，点击告警信息可回放告警时录像，支持按时间、告警类型、点位过滤检索，可进行负样本选择，告警数据存储 6 个月，实时预警延迟在 1s 以内。
告警数据收集：支持手动导出人工审核的误报样本，或选择训练平台自动打包上传。
即时告警显示：用户可配置接收特定类型告警，支持 web 页面弹出、邮件通知、气泡方式弹出告警消息（中文内容，含对象名称、时间、告警类型），点击告警信息弹出告警即时录像。
告警联动设置：支持设置联动规则并下发至智能分析板卡，支持三级预警模型（广播、声光、控制）。
联动预案设置：支持预先设定联动预案。
预警推送：支持将预警数据推送至邮件、APP 等平台。
日志中心功能
系统记录用户操作日志。
统计分析功能
针对不同维度的预警信息进行分类统计，输出统计数据。
API 接口功能
提供 RESTful/WebSocket 形式的 API 接口供第三方系统调用。
